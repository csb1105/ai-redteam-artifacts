# Reporting Guidelines

All reports must follow a consistent structure to ensure clarity and comparability.

## Required Sections
1. Summary of Findings  
2. Failure Mode Classification  
3. Evidence (transcript excerpts)  
4. Severity Assessment  
5. Contributing Factors  
6. Recommended Mitigations  
7. Follow‑Up Actions

## Evidence Requirements
- Include direct transcript excerpts
- Provide metadata references
- Maintain chain‑of‑custody for artifacts
